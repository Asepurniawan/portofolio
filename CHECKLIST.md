## Quick Reference

### Main Files to Edit
```
Portfolio Content → index.html
Styling → src/css/style.css
JavaScript Logic → src/js/script.js or src/js/modules/
```

### Important Paths
```
CSS: src/css/style.css
JS: src/js/script.js
Modules: src/js/modules/
Images: public/images/
```

### How Files Are Linked
```html
<!-- In index.html -->
<link rel="stylesheet" href="src/css/style.css">
<script src="src/js/script.js"></script>
<img src="public/images/202410370110122.jpg">
```
